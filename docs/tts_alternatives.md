## 2. OpenAI 的两套实时对话方案

如果您决定使用 OpenAI，需要根据“预算”选择不同的技术路线：

### 方案 A：OpenAI Realtime API (极致体验)
- **原理**: 类似于 ElevenLabs Agents。它是一个 **Speech-to-Speech** 模型，后端直接通过 WebRTC/WebSocket 传输音频，中间没有“文字”中转。
- **优点**: 延迟极低，支持打断，能听懂语气。
- **缺点**: **贵**。价格与 ElevenLabs 相当（约 $0.30/min）。

### 方案 B：OpenAI Standard TTS-1 (高性价比方案)
- **原理**: 经典的 **ASR -> LLM -> TTS** 管道。通过 `tts-1` 的流式接口实现边生成文字边播放语音。
- **优点**: **极便宜**！比 Realtime API 便宜约 **20倍**。
- **缺点**: 延迟略高（约 1s+），且需后端处理复杂的“流式拼接”。
