# AURA - AI 心理咨询师产品策略

## 1. 产品重新定位

**原定位**: 情感陪伴 AI → **新定位**: AI 心理咨询师

**一句话定义**: 随时在线的专业心理咨询师

**为什么心理咨询 > 陪伴**:
| 维度 | 陪伴型 | 心理咨询型 |
|------|--------|-----------|
| 用户认知 | "我不孤独，不需要" | "我有问题想解决" ✅ |
| 付费意愿 | 弱，娱乐属性 | 强，真人咨询300-800/次 ✅ |
| 使用场景 | 模糊 | 明确：焦虑/压力/人际/职场 ✅ |
| 竞品壁垒 | 低，很多聊天机器人 | 高，需要专业性 ✅ |

---

## 2. 目标用户

**核心用户画像**:
- 18-40岁城市白领/学生
- 有心理咨询需求但难以迈出第一步
- 因价格/时间/隐私顾虑未接受真人咨询
- 希望专业指导而非闲聊

**使用场景**:
1. 职场压力、工作焦虑
2. 人际关系困扰
3. 情绪管理（焦虑、抑郁倾向）
4. 重大决策迷茫
5. 亲密关系问题

---

## 3. 核心价值主张

> **"真人心理咨询的 1/10 价格，24小时在线，完全隐私"**

### 6.2 首次咨询的关键设计

#### 🎯 目标：让用户在3分钟内感受到价值

| 时间 | AI 行为 | 心理效果 |
|------|--------|---------|
| 0-30s | AI 主动开场，引用用户选择的问题 | "它知道我的困扰" |
| 30s-1min | 专业追问，引导用户表达 | "它在认真听我说" |
| 1-2min | 给出初步分析和共情 | "它理解我" |
| 2-3min | 提供具体建议或技巧 | "它能帮到我" |
| 结束时 | 总结要点 + 预告下次 | "期待下次对话" |

#### 🔊 首次咨询话术模板

**开场 (AI主动)**:
```
"你好，很高兴你愿意和我聊聊。
 我注意到你提到最近有[焦虑/压力/情绪低落]的感受，
 能和我说说，是什么事情让你有这样的感觉吗？"
```

**中间追问**:
```
"听起来[工作/关系/学业]给你带来了不少压力。
 这种感觉是最近才有的，还是已经持续一段时间了？"
```

**给出价值 (关键)**:
```
"从你描述的情况，我能感受到你承受了很多。
 这其实是一种很常见的[压力反应/焦虑表现]。
 
 我想分享一个简单的方法，叫做'5-4-3-2-1 接地法'，
 当焦虑来袭时，你可以：
 - 看见5样东西
 - 触摸4样东西
 - 听见3种声音
 - 闻到2种气味
 - 尝到1种味道
 
 这能帮助你快速回到当下。你愿意试试吗？"
```

**结束预告**:
```
"今天和你聊了这些，我记住了你最近的困扰。
 明天这个时候，我想问问你那个方法有没有帮助。
 记得，我24小时都在这里等你。"
```

### 6.6 关键流失节点干预

```
Day 0-1: 首次体验不佳
├── 问题: 不知道说什么 / AI 回复太机械
├── 对策: 主题引导 + 优化开场话术
└── 推送: "换个话题聊聊？试试[压力/情绪/人际]"

Day 2-3: 新鲜感消退
├── 问题: "聊了一次就够了"
├── 对策: 引用之前对话内容 + 解锁新功能
└── 推送: "上次你说[XX]，后来怎么样了？"

Day 4-7: 习惯未形成
├── 问题: 忘记使用 / 觉得没必要
├── 对策: 固定时间提醒 + 强化价值感知
└── 推送: "你的情绪比3天前稳定了12% 📈"
```

### 6.7 咨询对话风格指南

#### ❌ 避免 (太被动/泛泛)
- "嗯嗯，我在听"
- "你感觉怎么样？"
- "这确实挺难的"
- "你觉得应该怎么办？"

#### ✅ 推荐 (专业/有指导性)
- "听起来这件事给你带来了不少压力。能具体说说是什么环节让你最困扰吗？"
- "从心理学角度，这是一种典型的[认知偏差/应激反应]，很多人都有类似经历。"
- "我有一个具体的建议想分享给你：[具体方法]。你愿意试试吗？"
- "我注意到你已经连续3天提到工作压力，我们可以一起制定一个减压计划。"

## 8. 合规与风险

> ⚠️ **重要**: AI 不能替代真人心理治疗

**必须措施**:
1. 明确声明 "AI咨询仅供参考，不构成医疗建议"
